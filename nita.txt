git init = inicia una carpeta de seguimiento.
git status= estado del seguimiento.
git add . = guarda los cambios.
git commit -m "mensaje" = prepara los archivos guardados para subirlos.
git push origin master = sube los archivos
git clone URL = clona archivos de la nube
git log = historial de commit
git remote -v = verifica si se conecto local y nube.
git pull origin master= baja los cambios nuevo
**cambiar o poner informacion **
git config --global user.name = cambiar o poner informacion.
git config --global user.email = cambiar o poner correo.
git remote set-url origin = es para cambiar el url en caso que te equivoquen.
